Project Overview

RAT EOS PC delivers a low-cost IoT platform that detects fire-risk conditions in real-time:

  • Deployment of 34 LoRaWAN gateways
  • 600 sensors that capture biophysical fuel indicators
  • Solutions will be installed in Spain public utility mountains
  • Open-source stack technologies
  • Project website: www.rateospc.pt

System Architecture

  • Topology: Sensors → gateways → LoraWan servers → Mosquitto broker
  • Kafka buffers data
  • Logstash normalises according to FIWARE smart-data models.
  • Elasticsearch + Kibana power dashboards & maps.
  • N8N/ElastAlert Calculates fire risk alerts

Alerting Workflow

  1. Rules match in ElastAlert or N8N frameworks.
  2. Alert context enriched (geo info, sensor model).
  3. Stakeholders notified via Slack, e-mail & Kibana Dashboards
