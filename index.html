Early Warning System for Remote Sensing of Wildfires – Demo

Marta Oliveira, Luís Malarmey, Pedro Casau, Paulo Pedreiras, João Barraca, Pedro Fonseca (Univeristy of Aveiro, PT)
Sandra Millán Arias, Cristina Montesinos Barrios (CICYTEX, ES)

Project Overview

RAT EOS PC delivers a low-cost IoT platform that detects fire-risk conditions in real-time:

  • Deployment of 34 LoRaWAN gateways
  • 600 sensors that capture biophysical fuel indicators
  • Solutions will be installed in Spain public utility mountains
  • Open-source stack technologies
  • Project website: www.rateospc.pt

System Architecture

  • Topology: Sensors → gateways → LoraWan servers → Mosquitto broker
  • Kafka buffers data
  • Logstash normalises according to FIWARE smart-data models.
  • Elasticsearch + Kibana power dashboards & maps.
  • N8N/ElastAlert Calculates fire risk alerts

Alerting Workflow

  1. Rules match in ElastAlert or N8N frameworks.
  2. Alert context enriched (geo info, sensor model, fields measured).
  3. Stakeholders notified via Slack, Teams, e-mail & Kibana Dashboards

Contributions & Future Work

  • Scalable real-time wildfire monitoring for Mediterranean forests.
  • Entirely open-source and replicable in other regions.
  • Next Work: Integration with satellite images for multi-modal early warning and machine learning algorithms for predictive analytics.
